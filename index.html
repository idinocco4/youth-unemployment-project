Youth Unemployment Analysis

Predictive Analytics Project

Project Summary

This project explores youth unemployment trends using historical data and predictive analytics models. The analysis includes data exploration, modeling, forecasting, and an interactive dashboard.

Data Visualization

Key Findings

  • Youth unemployment rises during economic downturns.
  • Random Forest and XGBoost models effectively forecast trends.
  • Socioeconomic factors strongly influence unemployment rates.
