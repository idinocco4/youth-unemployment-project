# Youth Unemployment Project

This repository hosts a **predictive analytics project** analyzing youth unemployment trends using historical data. The project includes data exploration, predictive modeling, global trend analysis, and an interactive dashboard.

---

## üìä Project Overview

The goal of this project is to understand and forecast youth unemployment rates (ages 15‚Äì24) and examine key socioeconomic factors affecting employment trends. The analysis uses data from 1991‚Äì2024, including GDP growth, education enrollment, and labor force participation.

Key objectives:

- Explore historical trends in youth unemployment
- Identify correlations with socioeconomic indicators
- Build predictive models for forecasting
- Provide an interactive dashboard for visualization and exploration

## üóÇ Dataset

**Primary Dataset:** Youth Unemployment Data (1991‚Äì2024)  
**Features:**

| Feature | Description |
|---------|-------------|
| Year | Calendar year |
| Youth Unemployment Rate | Annual unemployment rate (%) |
